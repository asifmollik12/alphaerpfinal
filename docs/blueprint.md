# **App Name**: ModuleMart

## Core Features:

- Login/Signup: User authentication and registration form with secure password handling.
- Module Listing: Display available modules (HRM, CRM, Visa Management, Expense, POS, LMS) with individual pricing.
- Module Selection: Enable users to select and 'unlock' modules.
- Pricing Calculation: Calculate total price based on selected modules.
- Payment Gateway Integration: Integrate with Bangladeshi payment gateways (bKash, Nagad, Rocket, Payoneer).
- Dashboard Access: Grant access to selected module dashboards upon successful payment.
- AI-Powered Suggestion Tool: AI suggests which modules could benefit the user based on provided company information. This is implemented as a tool; the LLM may decide to not return anything if there is not enough information to make meaningful recommendations.

## Style Guidelines:

- Primary color: Light, desaturated blue (#A7D1AB) to evoke a sense of trust and efficiency, appropriate for a business application.
- Background color: Very light gray (#F0F0F0) for a clean and professional look.
- Accent color: A vibrant shade of violet (#456990) to highlight interactive elements.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, objective, neutral look that provides readability and a professional feel.
- Use clear and minimalist icons for module representation and actions.
- Maintain a clean and organized layout with intuitive navigation.
- Subtle animations for user interactions to enhance the user experience.
